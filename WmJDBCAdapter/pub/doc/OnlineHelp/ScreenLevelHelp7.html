Edit and View Schedule Screens

To create or modify a schedule for an adapter notification, provide values for the following parameters:

Field
Description

Interval (seconds)

Enter the polling interval time in seconds.

Overlap

Note: Do not use this option; otherwise, when you enable this notification, it may lock up tables and cause the Integration Server to fail.

Immediate

Enable this option to start polling immediately.

If you are viewing the schedule, the parameters are not editable.

To save the parameters you entered, click Save Schedule.

Click Return to JDBC Adapter Notifications to return to the Polling Notifications screen.
