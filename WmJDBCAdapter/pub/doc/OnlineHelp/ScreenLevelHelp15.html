ExecuteService Service Template Screens

An ExecuteService service allows a Java or flow service to use a connection from the adapter’s connection pool. For information about how a Java or flow service can use a connection, see the webMethods JDBC Adapter User’s Guide. This service template includes the following tab:

Service to Invoke Tab

Service to Invoke Tab

Select the Service to Invoke tab to specify the Java or the flow service name that needs the connection object:

Field
Description/Action

Service Name

The Java or the flow service name that requires to use a connection from the adapter’s connection pool. If you specify an incorrect service name, an exception is thrown at run time.

There are special considerations when creating a Java or flow service that uses a connection from the connection pool. For more information, see the webMethods JDBC Adapter User’s Guide.
