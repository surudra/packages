Connections Screen

Use the Connections screen to view and manage the status for all Oracle Applications Adapter connections currently configured on the Integration Server.

This screen shows the following information:

Connection Name
Displays the name of the connection.
Package Name
Displays the name of the package for the connection.
Connection Type
Displays the connection type associated with the connection. The Oracle Applications Adapter uses only the OracleApps Adapter Connection type.
Enabled
Indicates the availability of the connection specified in Connection Name. When enabled (Yes), the connection is available for IS services to use. When disabled (No), the connection is not available for IS services to use. You can only edit or delete a connection if it is disabled.

To enable a connection, click the No link in this field.

To disable a connection, click the Yes link in this field.

Edit
Displays the settings for the connection specified in Connection Name, and allows you to update the connection’s settings. You can only edit a connection if it is disabled.
View
Displays the settings for the connection specified in Connection Name. You cannot edit a connection from the View Connection screen.
Copy
Creates a new connection by copying the properties for an existing connection.
Delete
Deletes the connection specified in Connection Name. You can only delete a connection if it is disabled.

To create a new connection, click Configure New Connection.
