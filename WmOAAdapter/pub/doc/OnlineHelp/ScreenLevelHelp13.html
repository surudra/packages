Simple SQL Service Template

Use this template to configure Simple SQL services. Simple SQL services include Acknowledge services and Error services:

  • Acknowledge services maintain an accurate status of the transactions within the Oracle system. You create an Acknowledge service from a transaction definition that either calls a stored procedure or executes a SQL UPDATE statement to tell the Oracle Applications instance that the transaction has been processed.
  • Error services query Oracle Applications for errors that might have occurred during the import process. You create an Error service from a transaction definition that is based on a stored procedure, view, or SQL SELECT statement. The service receives the query parameters for the SQL statement via the service specification.

Specify template values using the Simple SQL tab.
