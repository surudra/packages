Edit and View Notification Schedule Screens

To create or modify a schedule for an adapter notification, provide values for the following parameters:

Field
Description

Interval (seconds)

Enter the polling interval time in seconds.

By default the next polling interval starts after the current execution is completed.

Overlap

Do not use this option; otherwise, when you enable this notification, it may lock up tables and cause the Integration Server to fail.

Immediate

Enable this option to start polling immediately.

By default the polling interval begins when the notification is enabled but enabling this option causes the notification to start immediately after it is enabled.

If you are viewing the schedule, the parameters are not editable.

To save the parameters yxou entered, click Save Schedule.

Click Return to Oracle Apps Adapter Notifications to go back to the Polling Notifications screen. Use this screen to enable the notification before you can use it.
