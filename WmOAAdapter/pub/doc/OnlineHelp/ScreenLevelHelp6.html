Polling Notifications Screen

The Polling Notification Screen allows you to schedule and manage adapter notifications.

From the Oracle Apps Adapter Polling Notifications table, use the fields in the following table to manage each adapter notification:

Field
Description

Notification Name

The name of the notification.

Package Name

The name of the package that contains the notification.

Enabled

After you schedule a polling notification, you can use this option to enable (Yes) or disable (No) the polling notification. Click on the current value in this field to change the value of this field.

If there is no polling notification scheduled for a given adapter notification, Not Scheduled appears in this field. Use the Edit Schedule icon to specify a schedule.

Note: You must schedule a polling notification before you can enable it.

Edit Schedule

Click on the Edit Schedule icon to create or modify polling notification parameters.

Note: You must disable a polling notification before you can edit it.

View Schedule

Click on the View Schedule icon to review the parameters for the selected polling notification. If the notification has not been scheduled, the icon is greyed out.

Click Return to Oracle Apps Adapter Notifications to go back to the Polling Notifications screen.
